Project Overview
About 3DSBank AI
Machine learning platform for banking intelligence — churn prediction and loan eligibility.

🤖 Models Used

  • ⚠️ ExtraTree Classifier — Churn (90.46%)
  • 🌳 Random Forest — Loan Eligibility (66.67%)
  • 🗳️ Voting Classifier — GaussianNB + AdaBoost + KNN
  • 📚 Stacking Classifier — RF + SVM + DT → LogReg
  • 🧠 Neural Network — Chatbot (53 intents)

🔧 Tech Stack

  • 🐍 Python 3.9 · pandas · numpy · sklearn
  • 🌲 Scikit-Learn — All ML classifiers
  • 🧠 TensorFlow / Keras — Chatbot model
  • 💬 NLTK — NLP & lemmatization
  • ⚖️ Imbalanced-learn — SMOTE / ROS
  • 📊 Seaborn / Matplotlib — Visualizations

📁 Project Files

M1.ipynb — Data Preprocessing (Churn) M2.ipynb — Data Visualization (Churn) M3-VOT-CH.ipynb — Voting + ExtraTree M5-RFC-LE.ipynb — Random Forest (Loan) M6-MULTI_MODEL-LE_.ipynb — Stacking M1_CB.ipynb / M2_CB.ipynb — Chatbot CHURN.pkl · LOAN_final1.pkl stacked_model.pkl churn.ipynb / loan.ipynb — Reports
